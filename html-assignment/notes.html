HTML Assignment | Description of 5 HTML Tags |
<img>:
this tage is used to display images on websites. It has attributes of src (source) and alt (alternative).
<p>:
This is a paragraph tag. It is used for structuring a story in different paragraphs similar to essay writing. It has a closing tag. </p>
<title>:
This is a title tag that gives name to a document. It has a closing tag. </title>
<body>:
This is a body tag that tells us about the structure of the document. It has a closing tag. </body>
<strong>:
This is a strong tag that makes an important text in a document bold. It has a closing tag. </strong>
